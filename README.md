# 🧠 Automated Fact Checker for Vernacular News

## 📌 Problem Statement
PS5 – Pipeline Optimization: Fact-Checker for Vernacular News

## 💡 Overview
This project detects fake news from regional languages using NLP.

## ⚙️ Features
- Supports Tamil, Hindi, English
- Uses translation + NLP classification
- Provides reliability score

## 🏗️ Architecture
Input → Translation → NLP Model → Output

## 🚀 How to Run
```bash
pip install -r requirements.txt
python app.py
